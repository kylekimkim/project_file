element.innerText="text node"

element.innerText : 해당 요소의 모든 자손의 text node를 문자열로 반환

    element.innerText="html text"

    element.innerHTML : 해당 요소의 자손의 HTML을 문자열로 반환

    문자열 안에 html 태그를 찾아서 node 개체로 생성

      document.write("html")

      document.write()가 선언된 시점에 node 개체 생성
