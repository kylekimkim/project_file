Number 기본 데이터 타입을 연산해 보자

operator

연산(operator) 이름 예시
+ add(더하기) 10.0+11.0=
- sub(빼기) 11.3-10.2=
* mult(곱하기) 10.1*20.2=
/ div(나누기) 20.0/2.0=
% mod(나머지) 21%4=
** Exponentiation(거듭제곱) 7**3=
++ increment(증가) ++10=
-- decrement(증감) --11=

할당 연산자

연산 예시 같은 연산
=(할당) x=y x가 y를 참조한다(x에 y를 할당한다.)
+= a+=b a=a+b
-= a-=b a=a-b
*= a*=b a=a*b
/= a/=b a=a/b
**= a**=b a**a-b
%= a%=b a=a%b

비교 연산자 a=10, b=12, c="10", d=10

비교 연산 후 boolean인 true or false 변환한다.

연산 의미 예시
a==b 동등 비교(형 변환 후 값을 비교)
a===b 엄격 동등 비교(타입까지 같아야 동등, type과 value 비교)
a!=b 비동등 비교(형 변환 후 value 비교))
a!==b 엄격 비동등 비교(type과 value 비교))
b>a 초과(형 변환 후 value 비교))
a < b 미만(형 변환 후 value 비교))
a <= b 이하(형 변환 후 value 비교))
