document.querySelector("선택자")

  • 선택자로 검색된 노드들 중에 첫번째로 검색된 노드를 반환
  • css 서낵자를 검색에 사용하고 document와 node 개체 모두 검색 가능

document.querySelectorAll("선택자")

  • 선택자로 검색된 복수의 노드를 NodeList로 반환
  • NodeList는 배열(array)처럼 인덱스가 있는 자료구조로 for of와 forEach를 사용할 수 있다.
