미디어 쿼리(@media())로 뷰포트 크기의 변화를 감지하자

화면의 크기를 변경해 보세요!

min-width:800px = 800px 이상일 때

max-width:700px = 700px 이하일 때

width:750px = 750px일 때

레이아웃이 max_width로 변경되는 html 참조(큰 모니터 css가 기본값

width:

  • default width > 1200px : 큰 모니터
  • @media (max-width: 1200px) : 모니터
  • @media (max-width: 992px) : 태블릿
  • @media (max-width: 768px) : 큰 폰(가로)
  • @media (max-width: 576px) : 작은 폰

레이아웃이 min_width로 변경되는 html 참조(작은 폰 css가 기본값

width:

  • default width > 576px : 작은 폰
  • @media (min-width: 576px) : 큰 폰
  • @media (min-width: 768px) : 태블릿
  • @media (min-width: 992px) : 모니터
  • @media (min-width: 1200px) : 큰 모니터
