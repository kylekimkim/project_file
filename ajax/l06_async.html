싱글 스레드와 멀티 스레드의 개념과 코드간의 동기와 비동기

싱글 스레드와 코드간 동기화

  • 거의 모든 프로그래밍 언어는 순차적 언어+싱글 스레드를 기본으로 제공한다.
  • single Tread : 한번에 코드 1개를 실행할 수 있다는 의미
  • 순차적 언어 : 프로그래밍이 싱글 스레드를 가지고 있어서 순서대로 1개씩 실행한다.
  • 코드간 동기 : 이전 코드가 실행을 완료해야 다음 코드가 실행된다.

멀티 스레드와 비동기

  • 스레드가 생성이 되면 멀티 스레드가 되고 스레드간의 코드는 동기화 되지 않는다.(비동기)
  • multi Tread : 한번에 스레드 개수 만큼 동시에 실행할 수 있다는 의미
  • 비순차적 언어 : 동시에 실행되는 코드간에 순서가 없다.
  • 코드간 비동기화 : 동시에 실행되는 코드간의 연관이 없음을 의미

멀티 스레드의 특징

  • 컴퓨터가 한꺼번에 많은 일을 하게 된다.
  • 일을 효율적으로 할 수 있다.(빠르게 일을 끝낸다.)
  • 컴퓨터가 너무 많은 일을 할 수도 있다.
